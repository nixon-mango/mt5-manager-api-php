## 🐳 Docker Installation & Usage

### Option 1: Docker Compose (Recommended)

1. **Clone and setup:**
   ```bash
   git clone https://github.com/dev4traders/mt5-manager-api.git
   cd mt5-manager-api
   
   # Copy environment template
   cp .env.example .env
   ```

2. **Configure your MT5 connection in `.env`:**
   ```env
   MT5_API_HOST=your-mt5-server.com
   MT5_API_PORT=443
   MT5_API_USERNAME=your-manager-username
   MT5_API_PASSWORD=your-manager-password
   ```

3. **Start the services:**
   ```bash
   # Production mode
   docker-compose up -d
   
   # Development mode (with Xdebug)
   docker-compose --profile dev up -d
   ```

4. **Run examples:**
   ```bash
   # Using the convenience script
   ./examples/docker_run_examples.sh all
   
   # Or run specific examples
   docker-compose exec mt5-sdk php /var/www/html/examples/basic_usage.php
   ```

### Option 2: Direct Docker Run

```bash
# Build the image
docker build -t mt5-manager-api .

# Run with environment variables
docker run --rm \
  -e MT5_API_HOST=your-mt5-server.com \
  -e MT5_API_USERNAME=your-username \
  -e MT5_API_PASSWORD=your-password \
  mt5-manager-api \
  php /var/www/html/examples/basic_usage.php
```

### Option 3: Development with Volume Mounts

```bash
# For active development
docker run --rm -it \
  -v $(pwd):/var/www/html \
  -e MT5_API_HOST=your-server.com \
  -p 8080:80 \
  mt5-manager-api bash
```

### Manual PHP Usage

```php
<?php
require_once(__DIR__ . '/vendor/autoload.php');

use D4T\MT5Sdk\MT5Manager\BasicApi;
use D4T\MT5Sdk\MT5Manager\AccountApi;
use D4T\MT5Sdk\Configuration;
use GuzzleHttp\Client;

// Configure API
$config = Configuration::getDefaultConfiguration();
$config->setHost('http://your-mt5-server.com/v1');

$client = new Client();
$basicApi = new BasicApi($client, $config);
$accountApi = new AccountApi($client, $config);

try {
    // Initialize and get token
    $initResult = $basicApi->initGet();
    $token = $initResult['token'];
    $config->setAccessToken($token);
    
    // Get account information
    $login = "12345"; // Your account login
    $result = $accountApi->accountLoginGet($login);
    print_r($result);
    
} catch (Exception $e) {
    echo 'Exception: ', $e->getMessage(), PHP_EOL;
}
?>
```

## 🐛 Troubleshooting

### Docker Build Issues

**PHP extension errors (json, curl, etc.):**
Try the builds in this order until one works:

```bash
# 1. Simplified build (removes multi-stage complexity)
docker-compose -f docker-compose.simple.yml up -d

# 2. Minimal build (only essential extensions)
docker-compose -f docker-compose.minimal.yml up -d

# 3. Automatic testing (finds what works)
./docker-test.sh
```

**Specific error fixes:**
- **json extension error**: Fixed - json is built into PHP 8.1+
- **libcurl error**: Use minimal build - includes libcurl4-openssl-dev
- **Multi-stage build issues**: Use simple or minimal builds

**Manual build testing:**
```bash
# Test minimal build directly
docker build -f Dockerfile.minimal -t mt5-test .
docker run -p 8080:80 mt5-test
```

### Docker Runtime Issues

**Containers won't start:**
```bash
# Check Docker status
docker info

# View container logs
docker-compose logs mt5-sdk

# Restart containers
docker-compose down && docker-compose up -d
```

**Permission issues:**
```bash
# Fix permissions
docker-compose exec mt5-sdk chown -R www-data:www-data /var/www/html
```

**Build cache issues:**
```bash
# Clear build cache and rebuild
docker-compose build --no-cache
# Or with simple version
docker-compose -f docker-compose.simple.yml build --no-cache
```

### API Connection Issues

**Connection refused:**
- Verify MT5 server is accessible
- Check firewall settings
- Confirm API is enabled in MT5 Manager

**Authentication failed:**
- Verify manager credentials
- Check manager permissions
- Ensure API access is enabled for the manager account

**Enable debug mode:**

```bash
# Or set debug environment
export XDEBUG_MODE=debug
docker-compose up -d
```
